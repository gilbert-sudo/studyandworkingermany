# Agency Information

## Goals
- Build scalable and robust MERN applications.
- Maintain high-quality code and project structure.

## Rules
- Keep the repository structured into `frontend`, `backend`, and `backup` directories.
- All backend routes should be under `/api`.
- Store configuration variables in `.env`.
- Write clear, maintainable code with proper comments.

## Operations
- Before significant structural changes, backup current stable state to `backup/`.
- Verify features completely before deployment.
